## 1. Introduction

### What is the Academic Advisor Module?

The Academic Advisor Module is a comprehensive system designed to help academic advisors guide students through their academic journey. It provides tools for:

- **Academic Planning:** Help students select courses and plan their academic pathway
- **Progress Monitoring:** Track student performance and identify at-risk students
- **Counseling:** Schedule and manage counseling sessions with students
- **Improvement Plans:** Create and track academic improvement strategies
- **Communications:** Send alerts, reminders, and notifications to students
- **Reports:** Generate comprehensive reports on student performance and advisor workload

### Who Can Use This Module?

- **Academic Advisors:** Primary users with full access to all features
- **School Administrators:** Can view and manage advisor activities
- **Slate Administrators:** Full system access

### Key Features

- ✅ Student academic profile management
- ✅ Course planning and recommendations
- ✅ At-risk student identification
- ✅ Progress report generation
- ✅ Counseling session management
- ✅ Improvement plan tracking
- ✅ Communication templates and logs
- ✅ Comprehensive analytics and reports

## 2. Getting Started

### Accessing the Module

1. Log in to SlateSMS with your academic advisor credentials
2. Select your school from the school selector
3. Navigate to **Dashboard** → **Academic Advisor**

### First-Time Setup

1. **Review Your Settings:**
   - Go to the **Settings** tab
   - Configure notification preferences
   - Set default report types
   - Adjust reminder settings

2. **Familiarize Yourself with the Dashboard:**
   - Review the Overview tab for key statistics
   - Check the Recent Activities feed
   - Explore each tab to understand available features

3. **Set Up Communication Templates:**
   - Go to **Communications** tab
   - Create templates for common messages
   - Customize templates with variables

## 4. Student Academic Profiles

### Accessing Student Profiles

1. Navigate to **Student Profiles** tab
2. Use the search bar to find students by name or ID
3. Filter by department, year, or program
4. Click on a student to view their detailed profile

### Student Profile Details

Each student profile includes:

#### Personal Information
- Name, Student ID
- Program/Degree
- Current Semester/Year
- Department

#### Academic Information
- **Course Enrollments:** Current and past courses
- **Grades:** Read-only view of student grades
- **GPA/SGPA/CGPA:** Current academic performance
- **Attendance Summary:** Overall attendance percentage

#### Risk Indicators
- **At-Risk Flags:** Active flags with severity levels
- **Low Attendance:** Attendance below threshold
- **Failing Grades:** Courses with failing grades

#### Advisor Actions

From the student profile, you can:

1. **Add Advisor Note:**
   - Click **Add Note**
   - Select note type (general, counseling, planning, performance, improvement)
   - Enter note content
   - Mark as confidential if needed
   - Add tags for easy searching

2. **Create Improvement Plan:**
   - Click **Create Improvement Plan**
   - Define improvement strategies
   - Set target GPA
   - Assign milestones

3. **Schedule Counseling Session:**
   - Click **Schedule Session**
   - Select session type
   - Choose date and time
   - Add session topic

4. **Upload Documents:**
   - Click **Upload Document**
   - Select document type
   - Upload file
   - Link to related plans or sessions

### Managing At-Risk Flags

1. **Create At-Risk Flag:**
   - Click **Flag as At-Risk** on student profile
   - Select flag type (low GPA, low attendance, failing courses, behavioral, other)
   - Set severity level (low, medium, high, critical)
   - Enter reason for flagging
   - Save flag

2. **Resolve At-Risk Flag:**
   - Open the flag details
   - Update status to "Resolved"
   - Add resolution notes
   - Save changes

## 5. Course Planning

### Academic Plans

#### Creating an Academic Plan

1. Navigate to **Course Planning** tab
2. Click **Academic Plans** sub-tab
3. Click **Create Plan**
4. Fill in the form:
   - **Student:** Select student
   - **Plan Type:** Choose type (4-year, 2-year, custom)
   - **Academic Year:** Enter academic year
   - **Current GPA:** Enter current GPA
   - **GPA Goal:** Set target GPA
   - **Graduation Year:** Expected graduation year
   - **Post-Secondary Goal:** Career or further education goals
   - **Career Interests:** Student's career interests
   - **Courses Planned:** List of planned courses
   - **Extracurricular Activities:** Relevant activities
   - **Notes:** Additional notes
5. Click **Save**

#### Managing Academic Plans

- **View Plan:** Click on a plan to view details
- **Edit Plan:** Click **Edit** to modify plan
- **Update Status:** Change plan status (active, completed, cancelled, on_hold)
- **Delete Plan:** Click **Delete** to remove plan

### Course Recommendations

#### Creating a Course Recommendation

1. Navigate to **Course Recommendations** sub-tab
2. Click **Create Recommendation**
3. Fill in the form:
   - **Student:** Select student
   - **Course:** Select or enter course code/name
   - **Recommendation Type:** Choose type (elective, required, remedial, bridge, prerequisite)
   - **Semester:** Target semester
   - **Academic Year:** Target year
   - **Reason:** Explain why this course is recommended
   - **Priority:** Set priority (high, medium, low)
4. Click **Save**

#### Managing Recommendations

- **View Recommendation:** Click to view details
- **Edit Recommendation:** Modify recommendation details
- **Update Status:** Change status (pending, approved, rejected, enrolled)
- **Delete Recommendation:** Remove if no longer needed

### Course Change Requests

#### Reviewing Course Change Requests

1. Navigate to **Course Change Requests** sub-tab
2. View list of pending requests
3. Click on a request to view details

#### Approving/Rejecting Requests

1. Open the request details
2. Review student's request and reason
3. Click **Approve** or **Reject**
4. Add comments explaining your decision
5. Save changes

## 6. Progress Monitoring

### At-Risk Student Flags

#### Creating an At-Risk Flag

1. Navigate to **Progress Monitoring** tab
2. Click **Create Flag**
3. Fill in the form:
   - **Student:** Select student
   - **Flag Type:** Choose type (low_gpa, low_attendance, failing_courses, behavioral, other)
   - **Severity:** Set severity (low, medium, high, critical)
   - **Flag Reason:** Explain why student is at-risk
   - **Flag Date:** Date of flagging
4. Click **Save**

#### Managing Flags

- **View Flag:** Click to view details
- **Update Status:** Change status (active, resolved, dismissed)
- **Add Resolution Notes:** Document how the issue was resolved
- **Delete Flag:** Remove if no longer needed

### Progress Reports

#### Creating a Progress Report

1. Navigate to **Progress Reports** page (via Progress Monitoring tab)
2. Click **Create Report**
3. Fill in the form:
   - **Student:** Select student
   - **Report Type:** Choose type (semester, annual, mid_term, final, custom)
   - **Academic Year:** Report year
   - **Semester:** Report semester
   - **Current GPA:** Current GPA
   - **Previous GPA:** Previous GPA (for comparison)
   - **Attendance Percentage:** Overall attendance
   - **Courses Completed:** Number of completed courses
   - **Courses In Progress:** Number of ongoing courses
   - **Courses Failed:** Number of failed courses
   - **Credits Earned:** Total credits earned
   - **Credits Attempted:** Total credits attempted
   - **Performance Summary:** Overall performance summary
   - **Strengths:** Student's academic strengths
   - **Areas for Improvement:** Areas needing improvement
   - **Recommendations:** Recommendations for student
   - **Next Steps:** Action items for student
4. Click **Save**

#### Finalizing Reports

1. Review the report details
2. Click **Finalize Report** when ready
3. Once finalized, the report cannot be edited
4. Generate PDF by clicking **Generate PDF**

#### Viewing and Managing Reports

- **Search Reports:** Use search bar to find reports
- **Filter by Type:** Filter by report type
- **Filter by Status:** Show finalized or draft reports
- **View Report:** Click to view full report details
- **Edit Report:** Edit draft reports
- **Delete Report:** Remove if needed

## 12. Best Practices

### Academic Planning

1. **Create Plans Early:** Create academic plans at the beginning of each semester
2. **Review Regularly:** Review and update plans periodically
3. **Set Realistic Goals:** Set achievable GPA and graduation goals
4. **Document Everything:** Keep detailed notes on all planning discussions

### Progress Monitoring

1. **Flag Early:** Flag at-risk students as soon as issues are identified
2. **Set Appropriate Severity:** Use severity levels appropriately
3. **Follow Up:** Regularly follow up on flagged students
4. **Document Resolutions:** Always document how issues were resolved

### Counseling Sessions

1. **Prepare in Advance:** Review student profile before sessions
2. **Take Detailed Notes:** Document all important points discussed
3. **Set Follow-ups:** Schedule follow-up sessions when needed
4. **Respect Confidentiality:** Mark sensitive notes as confidential

### Improvement Plans

1. **Be Specific:** Define clear, measurable improvement strategies
2. **Set Milestones:** Break down plans into manageable milestones
3. **Track Progress:** Regularly update completion percentage
4. **Review Frequently:** Conduct reviews as per review frequency

### Communications

1. **Use Templates:** Create and use templates for common messages
2. **Personalize Messages:** Use variables to personalize communications
3. **Time Appropriately:** Send reminders at appropriate times
4. **Track Responses:** Monitor communication logs for responses

## 13. Troubleshooting

### Common Issues

#### Cannot Access Module

**Problem:** Cannot see Academic Advisor dashboard

**Solutions:**
- Verify your role is set to "academic-advisor"
- Contact system administrator to assign role
- Check if you have proper school access

#### Cannot View Student Data

**Problem:** Student profiles are not loading

**Solutions:**
- Verify student belongs to your school
- Check if student status is "active"
- Refresh the page
- Clear browser cache

#### Cannot Create Reports

**Problem:** Reports are not generating

**Solutions:**
- Check if you have proper permissions
- Verify data exists for the selected filters
- Try a different date range
- Contact support if issue persists

#### Communication Not Sending

**Problem:** Messages are not being sent

**Solutions:**
- Verify communication service is configured
- Check recipient email/phone numbers
- Review communication logs for errors
- Contact IT support

### Getting Help

- **System Administrator:** Contact for role/permission issues
- **IT Support:** Contact for technical issues
- **Module Documentation:** Refer to this manual
- **Training Resources:** Access training materials if available

## 14. FAQs

### General Questions

**Q: Can I edit grades?**  
A: No, academic advisors have read-only access to grades. Only authorized personnel can modify grades.

**Q: Can I access other schools' data?**  
A: No, you can only access data for your assigned school. Multi-tenant security ensures data isolation.

**Q: How do I reset my password?**  
A: Contact your system administrator or use the password reset feature on the login page.

### Academic Planning

**Q: Can I create multiple plans for one student?**  
A: Yes, you can create multiple plans, but typically one active plan per student is recommended.

**Q: How do I approve course change requests?**  
A: Navigate to Course Planning → Course Change Requests, review the request, and click Approve or Reject.

### Progress Monitoring

**Q: What's the difference between a flag and an improvement plan?**  
A: A flag identifies an at-risk student, while an improvement plan provides a structured approach to address the issues.

**Q: Can I delete a finalized progress report?**  
A: No, finalized reports cannot be deleted to maintain data integrity. You can create a new report if needed.

### Communications

**Q: Can I send bulk messages?**  
A: Yes, you can select multiple students or groups when sending communications.

**Q: How do I create custom templates?**  
A: Navigate to Communications → Templates, click Create Template, and fill in the form with your custom content.

### Reports

**Q: How often should I generate reports?**  
A: Generate reports as needed. Weekly or monthly reports are common for tracking progress.

**Q: Can I schedule automatic reports?**  
A: Currently, reports must be generated manually. Automatic generation may be available in future updates.
